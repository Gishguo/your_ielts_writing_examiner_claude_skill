# IELTS Writing Grading Skill for Claude Code | Claude Code IELTS 写作评分技能

An AI-powered IELTS Writing assessment skill that runs locally in Claude Code, grading essays against the official IELTS Writing Band Descriptors (updated May 2023).

一个基于 Claude Code 的 IELTS 写作评分技能，在本地运行，严格依据官方 IELTS 写作评分标准（2023年5月更新）对作文进行评分。

---

## Features | 功能

- Grades both Task 1 and Task 2 (Academic & General Training) | 支持 Task 1 和 Task 2（学术类 & 培训类）
- Scores all four criteria with half-band precision | 四项评分标准均支持半分段评分
- Produces structured feedback with specific evidence | 生成结构化反馈，引用具体文本证据
- Suggests priority actions for improvement | 提供优先改进建议
- Optionally provides a revised version highlighting changes | 可选提供修改版本并标注改动

## Quick Start | 快速开始

1. Install [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | 安装 [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
2. Clone this repo into your project or copy the `.claude/skills/` directory | 克隆此仓库或复制 `.claude/skills/` 目录到你的项目
3. Open Claude Code in the project directory and ask it to grade your essay | 在项目目录中打开 Claude Code，要求它为你的作文评分

Example prompt | 示例提示：

```
Grade this IELTS Task 2 essay: [paste your essay here]
```

## How It Works | 工作原理

The skill follows a five-step grading process | 评分流程分为五步：

1. **Identify task type** — Task 1 vs Task 2, Academic vs General Training | 识别任务类型
2. **Analyse the essay** — Evaluate against all four IELTS criteria | 分析作文
3. **Score each criterion** — Using official band descriptors, with half-bands | 按官方评分标准逐项打分
4. **Calculate overall band** — Arithmetic mean rounded to nearest half band | 计算总分
5. **Produce report** — Structured feedback with strengths, areas for improvement, and priority actions | 生成报告

## Scoring Criteria | 评分标准

| Criterion | Description |
|-----------|-------------|
| Task Achievement / Task Response | 任务完成情况 / 任务回应 |
| Coherence & Cohesion | 连贯与衔接 |
| Lexical Resource | 词汇资源 |
| Grammatical Range & Accuracy | 语法多样性与准确性 |
